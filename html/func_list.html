Soruto EJBuilder 関数リスト

関数のリストです。
SEJBの関数には他の関数との競合を防ぐため最初に必ず So_ がついています
Ctrl+Fを使うと簡単に見つけられます

So_al(string)
alert()と同じ働きをする。
e.g. So_al("Hello World");//Hello Worldをアラート表示

So_gId(id)
getElementById()と同じ働きをする。
e.g. So_gId("st").value = "Hello World";//id=st のvalueをHello Worldにする

So_cl(string)
console.log()と同じ働き。
e.g. So_cl("Hello World")//ブラウザコンソールにhello worldを表示

So_SetVal(id,string)
第1引数に指定されたidの要素のvalueに第2引数の文字列をセット
e.g. <script>So_SetVal("test","Hello World");//idがtestのinputのvalueをHello Worldにする
</script>
<input type="text" id="test">

So_GetVal(id)
引数に指定されたidのvalueの値を返します
e.g. <script>var st = SoGetVal("st");
So_al(st);//Hello!!がアラートに表示される</script>
<input type="text" value="Hello!!">
